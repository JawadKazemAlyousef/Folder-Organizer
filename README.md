	<h1>منظم الملفات</h1>
	<h2>الغرض من منظم الملفات تنظيم ملفات المجلد المختار حسب ثلاثة أوضاع:</h2>
 	<h3>1. الوضع العادي: نظم ملفات المجلد المختار حسب صيغهم.</h3>
 	<h4>التعليمات:</h4>
	<h5>
1. انقر زر العادي.<br><br>
2. انقر زر اختر لتختار المسار (موقع المجلد).<br><br>
3. اختر المسار (موقع المجلد) الذي تريد أن ترتبه.<br><br>
4. تأكد من أنك حفظت الإعدادات بهذا الوضع (بنافذة الوضع العادي أو النافذة الرئيسة) وهذا عن طريق نقر قائمة ملف، ثم خيار حفظ أو حفظ كـ إذا كنت تريد أن تحفظ الإعدادات بتخزينة جديدة.<br><br>
5. انقر زر نظم لتنظم ملفات المجلد المختار حسب صيغهم.<br><br>
	</h5>
	<h3>2. الوضع الموصى: ينظم ملفات المجلد المختار حسب ما استحسنه</h3>
  	<h4>التعليمات:</h4>
   	<h5>
1. انقر زر الموصى.<br><br>
2. انقر زر اختر لتختار المسار (موقع المجلد).<br><br>
3. اختر المسار (موقع المجلد) الذي تريد أن ترتب ملفاته.<br><br>
4. اختر اللغة التي تريد أن تكون أسماء المجلدات بها.<br><br>
5. تأكد من أنك حفظت الإعدادات بهذا الوضع (بنافذة الوضع الموصى) وهذا عن طريق نقر قائمة ملف، ثم خيار حفظ أو حفظ كـ إذا كنت تريد أن تحفظ الإعدادات بتخزينة جديدة.<br><br>
6. انقر زر نظم لتنظم ملفات المجلد المختار حسب ما استحسنه.<br><br>
ملاحظة: إذا كان هناك صيغ ملفات غير موجودة في قائمة الموصى بها، سترتب تلقائيا حسب صيغهم.<br><br>
	</h5>
	<h3>3. الوضع المخصص: ينظم ملفات المجلد المختار حسب ما تخصصه.</h3>
  	<h4>التعليمات:</h4>
	<h5>
1. انقر زر المخصص.<br><br>
2. انقر زر اختر لتختار المسار (موقع المجلد).<br><br>
3. اختر المسار (موقع المجلد) الذي تريد أن ترتب ملفاته.<br><br>
4. أنت تستطيع تخصيص تنظيمك يدويا، أو رفع تخزينتك المخصصة.<br><br>
أ. بالنقر على زر خصص أنت تستطيع تخصيص تنظيمك يدويا.<br><br>
تأكد من نقر حسنا لتحديث تغيراتك الجديدة.<br><br>
ب. بالنقر على زر ارفع،  أنت تستطيع رفع تخزينتك المخصصة.<br><br>
تأكد من اختيار تخزينتك بصيغة .ifoi ويكون المحتوى كهذا :<br><br>
اسم_المجلد:<br><br>
{اسم_الصيغة, اسم_الصيغة}<br><br>
اسم_المجلد:<br><br>
{اسم_الصيغة}<br><br>
اسم_المجلد:<br><br>
{}<br><br>	
اسم_المجلد يجب أن يكون مسموحا، وممكن أن يكون من كلمات متعددة.<br><br>
اسم_الصيغة يجب أن يكون مسموحا دون نقطة (.)، وممكن أن يكون من كلمات متعددة.<br><br>
5. تأكد من أنك حفظت الإعدادات بهذا الوضع (بنافذة الوضع المخصص) وهذا عن طريق نقر قائمة ملف، ثم خيار حفظ أو حفظ كـ إذا كنت تريد أن تحفظ الإعدادات بتخزينة جديدة.<br><br>
6. انقر زر نظم لتنظم ملفات المجلد المختار حسب ما تخصصه.<br><br>
ملاحظة: إذا كان هناك صيغ ملفات غير موجودة في قائمتك المخصصة، سترتب تلقائيا حسب صيغهم.
	</h5>

	<h1>Folder Organizer</h1>
	<h2>The purpose of the program is to organize the files of the chosen folder based on three modes:</h2>
 	<h3>1. Standard mode: Organize the files of the chosen folder based on their extension formats.</h3>
 	<h4>Instruction:</h4>
	<h5>
1. Click Standard button.<br><br>
2. Click Choose button to select the path (the folder location).<br><br>
3. Choose the path (the folder location) that you want to organize.<br><br>
4. Make sure to save the settings at this specific mode (at the same window or the main window) by clicking File Menu, and then Save Option or Save As option if you want to save in the new saved setting.<br><br>
5. Click Organize button to organize the folder's files based on their extension formats.<br><br>
	</h5>
	<h3>2. Recommended mode: Organize the files of the chosen folder based on my recommended choice.</h3>
 	<h4>Instruction:</h4>
   	<h5>
1. Click Recommended button.<br><br>
2. Click Choose button to select the path (the folder location).<br><br>
3. Choose the path (the folder location) that you want to organize.<br><br>
4. Choose the language that you want folders' names based on.<br><br>
5. Make sure to save the settings at this specific mode (at the same window or the main window) by clicking File Menu, and then Save Option or Save As option if you want to save in the new saved setting.<br><br>
6. Choose the language that you want folders' names based on my recommended choice.<br><br>
Note: If there are files' extension formats that are not in the recommended mode list, they will be automatically organized based on their extension formats.<br><br>
	</h5>
	<h3>3. Customized mode: Organize the files of the chosen folder based on your customized choice</h3>
 	<h4>Instruction:</h4>
	<h5>
1. Click Customized button.<br><br>
2. Click Choose button to select the path (the folder location).<br><br>
3. Choose the path (the folder location) that you want to organize.<br><br>
4. You can either manually customize your organization or upload your customized saved file.<br><br>
				
  A. By clicking Customize button, you can manually customize your organization.<br><br>
     Make sure to click OK button to update the new changes.<br><br>
  B. By clicking Upload button, you can upload a customized saved file.<br><br>
     Make sure to choose the saved file in .ifoi extension format and the contents be something like:<br><br>
       Folder_name:<br><br>
       {Extension_name, Extension_name}<br><br>
       Folder_name:<br><br>
       {Extension_name}<br><br>
       Folder_name:<br><br>
	{}<br><br>
Folder_name must be a valid name, and it can be multiple words.<br><br>
Extension_name must be a valid name without a dot (.), and it can be multiple words.<br><br>
5. Make sure to save the settings at this specific mode (at the same window or the main window) by clicking File Menu, and then Save Option or Save As option if you want to save in the new saved setting.<br><br>
6. Choose the language that you want folders' names based on your customized choice.<br><br>
Note: If there are files' extension formats that are not in your customized list, they will be automatically organized based on their extension formats.<br><br>
	</h5>
